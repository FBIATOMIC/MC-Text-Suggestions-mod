# MC-Text-Suggestions-mod
Unlocks Windows text suggestions directly inside Minecraft chat.

MC Text Suggestions: To make this mod work, you need to have Windows Text Suggestions enabled. If you don't have it enabled, do win + R paste "ms-settings:typing" then turn ON the appropriate option mentioned below according to your OS version:

Windows 10: "Show text suggestions as I type"  {TURN IT ON}
Windows 11: "Show text suggestions when typing on the physical keyboard"  {TURN IT ON}"

You can trun ON/OFF the mod useing F12
